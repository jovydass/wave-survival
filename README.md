## How the game works

The player controls a character that can move around the arena and shoot enemies. Enemies spawn in waves and become more difficult over time.

The player gains experience points by defeating enemies. When enough experience is collected, the player levels up and can choose an upgrade such as increased health, faster shooting, or higher movement speed.

The game ends when the player’s health reaches zero. The player can restart the game and try to achieve a higher score.
